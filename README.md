# walpp

A rewrite of __MOST__ of [pywal's](https://github.com/dylanaraps/pywal) functionality using [QT](https://www.qt.io/).
***Only works on Linux at the moment.***.

### Why/Philosophy

Due to Arch Linux's fast-moving rolling updates nature, it was always tiring to re-install Pywal with every new Python version.
I, therefore decided to attempt to rewrite it using C++ and Qt seemed like a viable option.
Having a C++ alternative means that I have an executable that'll last through OS updates.
